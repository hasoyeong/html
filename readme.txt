선택자 {속성명:속성값;}  -속성명은 파란색글씨로된것, 속성값은 주황색글씨로된것, 속성값은 단위를 적는 것도 있고 안 적어도 되는 것도있는데 CSS에서는 꼭 적어야 됨. 속성명과 속성값을 같이 알아두면 좋음

선택자
- 전체선택자(*) -태그로 구성된 모든 요소들을 지정
- 태그(요소)선택자 -태그 이름으로 지정
- 클래스선택자 - 클래스이름으로 지정, 이름을 직접 지을수있고 가장 많이쓰는 선택자임
- 아이디선택자 - 한번만 쓸 수 있음
- 그룹선택자 - ex: html, body {wheit:50px }
- 하위선택자(자손) -상위에 대해서 밑에있는 모든 녀석들 
- 하위선택자(자식) -상위에 대해서 바로 밑에 있는 녀석들, 밑에밑에는 자식이 아니라 자손들
- 속성 선택자


글꼴관련 스타일
font-family -글꼴(글자의 모양) 사용할 때는 꼭 저작권(라이선스)를 확인. 사용법: 눈누에서 사용하고싶은 폰트  stly이라는 태그 안에 넣고 적용하고싶은 선택자 안에 font-family 'ONE-Mobile-POP'; 를 적어주면 됨
font-size - px, em, rem, cm, mm, inch, % (1px은 점한개 정도)
 16px = 1em 또는 1rem
 em, rem은 기본적 크기는 같은데 소속이 되어있는 범위에 따라 크기가 달라질 수 있음
 12pt != 12px 포인트는 인쇄단위의 크기라서 mm, cm, inch 등과 같이 사용
72pt = 1inch = 2.54cm
font-wheit : 굵기 100 ~ 900, thin ~ bolder, 기본값 normal : 400
font-stly - 기울임꼴 italic
